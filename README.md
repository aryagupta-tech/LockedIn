# LockedIn Landing Page

Production-ready Next.js 15 App Router landing page for **LockedIn** (`joinlockedin.social`).

## Stack

- Next.js 15 + TypeScript
- Tailwind CSS
- shadcn-style UI components
- Framer Motion
- Lucide React
- React Hook Form + Zod

## Run locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Build

```bash
npm run build
npm start
```

## Notes

- Dark-mode only cyber-minimal aesthetic
- SEO metadata + Open Graph + JSON-LD included
- Fully responsive sections and interactive gate demo modal
